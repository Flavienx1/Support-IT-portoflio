# Portfolio Support IT N1

Portfolio de mise en pratique orienté support IT niveau 1.

Ce portfolio regroupe des cas pratiques liés au diagnostic d’incidents utilisateurs, au dépannage Windows, au réseau, à Microsoft 365, à Active Directory / Entra ID et aux commandes PowerShell de base.

## Objectif

Montrer ma capacité à analyser un incident, appliquer une démarche de diagnostic structurée, utiliser les bons outils et documenter les actions réalisées.
